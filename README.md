# textarea Counter
This is a Jquery plugin for counter on textArea input form
## Settings
        limit: 500,
        startCounter: 0,
        textMax: "Max",
        textCaracter: "Counter",
        titleColor: "#1578AD",
        textColor: "#737577",
        textAlign: "right",      //left|center|right
        width: 500,              //same textarea width
        marginLeft: 10           //set margin left for counter
